# 番茄风/网文风 提示词重构 (2026-02-05)

## 背景
用户反馈目前的“雪花写作法”过于学院派，无法写出“番茄小说”的爽感和味道。需要重构核心提示词，引入网文核心概念（金手指、爽点、黄金三章）。

## 目标
- 将架构生成逻辑从“文学构建”转向“网文爽感构建”。
- 强化角色设定的“金手指”和“仇恨/期待”属性。
- 剧情结构改为“黄金三章”+“打脸循环”。
- 章节写作强调节奏快、钩子（Cliffhanger）和情绪宣泄。

## 任务分解
- [ ] 重构架构提示词 `src/prompts/architecture.js` <!-- id: fanqie-01 -->
    - [ ] 核心种子：加入金手指、脑洞、爽点要素
    - [ ] 角色设定：强调主角外挂、反派降智/拉仇恨、配角工具人属性
    - [ ] 情节架构：引入“黄金三章”和“升级流”结构
- [ ] 重构章节提示词 `src/prompts/chapter.js` <!-- id: fanqie-02 -->
    - [ ] 强化“每章一爽”或“每章一钩子”
    - [ ] 强调情绪调动（震惊、期待、愤怒）
