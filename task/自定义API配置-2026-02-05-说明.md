# 自定义 API 配置任务说明

**日期**: 2026-02-05
**执行者**: 左岚 (Executor)
**状态**: 进行中

## 任务目标
支持通过环境变量配置自定义 API 地址和密钥。

## 变更内容
1.  **新增 `.env`**: 存储 `VITE_AI_BASE_URL`, `VITE_AI_API_KEY`, `VITE_AI_MODEL`。
2.  **修改 `src/stores/settings.js`**: 读取环境变量作为默认配置。

## 验证方法
1.  检查 `.env` 文件是否存在且内容正确。
2.  启动应用，检查设置面板中的 API 配置是否自动填充。
